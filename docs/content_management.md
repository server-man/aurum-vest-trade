# Content Management Guide

[Content moved from root CONTENT_MANAGEMENT_GUIDE.md]

This guide covers content management features in AurumVest.

## CMS Features
- Email template management
- Notification management
- Dynamic content updates

## Admin Access
Admins can access content management through:
- `/admin/content-management` route
- CMS Manager component
- Email template editor

## Content Types
1. **Email Templates** - Transactional emails
2. **Notifications** - In-app notifications
3. **Marketing Content** - Landing page content

## Editing Content
Content can be edited through the admin dashboard without code changes.
